2. System Architecture
Testing Dashboard (Frontend 1): Upload videos/images, sends frames via WebSocket,
displays output.
Backend + AI Engine: FastAPI + YOLOv8, processes frames, detects hazards, stores data.
Admin Panel (Frontend 2): Real-time alerts and analytics for authorities.
3. End-to-End Workflow
•
User uploads video
•
Frames extracted and sent via WebSocket
•
AI processes frames using YOLOv8
•
Severity and risk calculated
•
Frames annotated with OpenCV
•
Processed frames returned
•
Alerts triggered if severity high
•
Admin panel updates in real-time
4. Data Flow & Contracts
Frontend → Backend
{ "frame": "base64String", "timestamp": 1.5 }
Backend → Testing Dashboard

Backend → Admin Panel
